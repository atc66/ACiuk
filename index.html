DevConnect

DevConnect is a web based application designed to connect aspiring developers. The backend API ws built with Node.js and Express and was tested with Postman. Routes are protected with JSON Web Tokens. The frontend was built to work the API, and integrate React with the backend. Redux was used for app state management, and it was deployed to Heroku with a postbuild script.


FindWater

FindWater is a web based application that I decided to write after training for the Philadelphia Half-Marathon. When running I never carry cash, credit cards, or a water bottle. So I either finish my runs in a distance where I don't need a drink of water, or I plan my routes around where I know there are water fountains. But what if there was a resource that mapped locations of where you can get free water? FindWater was written with Ruby on Rails, and integrates the Google Maps API allowing users to store locations on the map of where to get water.


SportsTracker

Sports Tracker was a collaborative project, that clones sports webpages but allows users to submit photos of themselves at the game. A social media-based platform for sports! SportsTracker was written with Ruby on Rails, and integrated the mysportsfeed RESTful API to gather the statistical information about the chosen event.
